Improve the editor input area with placeholder text, better model/thinking visibility, and contextual hints.

## Current state
- Editor is a bordered box with no content when empty — blank and uninviting
- Model name only appears in status bar (bottom right, easy to miss)
- Thinking level shown as border color (very subtle)
- No hint about keybindings for new users

## Changes to `views/editor.rs`

### 1. Placeholder text when editor is empty

In `EditorView::render()`, after rendering the block, if content is empty:

```rust
impl Widget for EditorView<'_> {
    fn render(self, area: Rect, buf: &mut Buffer) {
        // ... existing block + content rendering ...
        
        // Placeholder when empty (render AFTER paragraph so it doesn't conflict)
        if self.state.content.is_empty() && !self.is_streaming {
            let placeholder = "Ask anything… ⇧↵ newline • @file • /commands";
            let placeholder_style = Style::default().fg(Color::DarkGray);
            buf.set_string(
                inner.x,
                inner.y,
                placeholder,
                placeholder_style,
            );
        }
    }
}
```

### 2. Model + thinking in editor title

Currently the title shows "streaming…" during streaming and nothing otherwise. Add model + thinking level to the right side of the border:

```rust
let title = if self.is_streaming {
    if self.has_queued {
        " streaming [queued] ".to_string()
    } else {
        " streaming… ".to_string()
    }
} else {
    String::new()
};

// Right-side title: model + thinking level
let thinking_label = match self.thinking_level {
    ThinkingLevel::Off => "",
    ThinkingLevel::Minimal => "min",
    ThinkingLevel::Low => "low",
    ThinkingLevel::Medium => "med",
    ThinkingLevel::High => "high",
    ThinkingLevel::XHigh => "xhigh",
};
let right_title = if thinking_label.is_empty() {
    format!(" {} ", self.model_name)
} else {
    format!(" {} • {} ", self.model_name, thinking_label)
};
```

Use ratatui's `Block::title_bottom()` or `Block::title()` with alignment for the right title. If ratatui version supports it:
```rust
let block = Block::default()
    .title(title)
    .title_bottom(Line::from(right_title).alignment(Alignment::Right))
    .borders(Borders::ALL)
    .border_style(Style::default().fg(border_color));
```

If `title_bottom` isn't available, render the right title manually by writing to the bottom-right of the border area.

### 3. Add model_name to EditorView

The `EditorView` needs to know the model name. Add it:

```rust
pub struct EditorView<'a> {
    state: &'a EditorState,
    theme: &'a Theme,
    thinking_level: ThinkingLevel,
    model_name: &'a str,       // ADD
    is_streaming: bool,
    has_queued: bool,
}

impl<'a> EditorView<'a> {
    pub fn new(state: &'a EditorState, theme: &'a Theme, thinking_level: ThinkingLevel) -> Self {
        Self {
            // ...
            model_name: "",
            // ...
        }
    }
    
    pub fn model(mut self, name: &'a str) -> Self {
        self.model_name = name;
        self
    }
}
```

### 4. Update editor construction in app.rs

In `App::render()`:
```rust
let editor = EditorView::new(&self.editor, &self.theme, self.thinking_level)
    .model(&self.model_name)    // ADD
    .streaming(self.is_streaming)
    .queued(!self.message_queue.is_empty());
```
