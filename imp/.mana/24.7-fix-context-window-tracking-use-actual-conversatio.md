The context percentage in the status bar is wrong. It shows cumulative tokens / context_window, which grows without bound and doesn't represent actual context usage.

## Current behavior (app.rs)

```rust
// In build_status_info():
let total_input = self.accumulated_usage.input_tokens;
let total_output = self.accumulated_usage.output_tokens;
let context_used = (total_input + total_output) as f64;
let context_percent = if self.context_window > 0 {
    context_used / self.context_window as f64
} else {
    0.0
};
```

This accumulates across ALL turns. After 3 turns with 50k input each, it shows 150k/200k = 75% — but the actual conversation might only be 60k tokens because compaction happened.

## What it should track

The **last turn's input_tokens** is the best proxy for current context size. The input tokens on each API call represent the full conversation sent to the model.

### Changes to `App` struct

Replace `accumulated_usage` approach for context tracking:
```rust
pub struct App {
    // ... existing ...
    pub accumulated_usage: Usage,     // KEEP for total display
    pub accumulated_cost: Cost,       // KEEP
    pub current_context_tokens: u32,  // ADD: last turn's input tokens
}
```

Initialize `current_context_tokens: 0` in `App::new()`.

### Update AgentEnd handler

In `handle_agent_event`:
```rust
AgentEvent::AgentEnd { usage, cost } => {
    self.accumulated_usage.add(&usage);
    self.accumulated_cost.total += cost.total;
    self.accumulated_cost.input += cost.input;
    self.accumulated_cost.output += cost.output;
    
    // Track actual context size from this turn's input
    self.current_context_tokens = usage.input_tokens;  // ADD
    
    // ... rest of handler ...
}
```

Also update on `TurnEnd` if that carries usage info, or on `AgentStart` reset.

### Update build_status_info

```rust
fn build_status_info(&self) -> StatusInfo {
    // ...
    let context_percent = if self.context_window > 0 {
        self.current_context_tokens as f64 / self.context_window as f64
    } else {
        0.0
    };
    // ...
}
```

### Update after compaction

In `AgentEvent::CompactionEnd`:
```rust
AgentEvent::CompactionEnd { summary } => {
    // Context shrunk — reset tracking (next turn will update)
    self.current_context_tokens = 0;
    // ... existing code ...
}
```

### Status bar color coding

In `views/status.rs`, color the context percentage based on usage:
- < 50%: muted (normal)
- 50-75%: yellow (warning)
- 75%+: red (danger, compaction soon)

```rust
let context_style = if self.info.context_percent > 0.75 {
    self.theme.error_style()
} else if self.info.context_percent > 0.50 {
    self.theme.warning_style()
} else {
    self.theme.muted_style()
};
// Use context_style instead of muted_style for the context% span
```
