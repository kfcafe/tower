Wire the imp-core `UserInterface` trait to the TUI so the agent can request approval before destructive operations. Currently the TUI removes the `ask` tool entirely because it's not wired.

## Current state

In `app.rs::spawn_agent_for_prompt()` (around line 795):
```rust
// Remove ask tool — TUI doesn't wire UserInterface to the agent yet
agent.tools.retain(|name| name != "ask");
```

The agent has a `UserInterface` trait that supports `ask()` and `confirm()`. The TUI needs to implement this via channels so the agent can pause and wait for user input.

## Architecture

The agent runs on a background tokio task. The TUI runs on the main thread with a crossterm event loop. Communication happens via channels (AgentEvent for agent→TUI, AgentCommand for TUI→agent). We need a new bidirectional channel for approval requests.

### 1. Define approval types

Add to `app.rs` (or a new `approval.rs` module):

```rust
use tokio::sync::oneshot;

pub struct ApprovalRequest {
    pub question: String,
    pub options: Vec<String>,
    pub tx: oneshot::Sender<ApprovalResponse>,
}

pub struct ApprovalResponse {
    pub selected: Option<usize>,  // None = dismissed/cancelled
}
```

### 2. Add approval channel to App

pub struct App {

### 3. Implement UserInterface for a TUI bridge

Create a struct that implements `UserInterface` and sends requests through the channel:

```rust
pub struct TuiUserInterface {
    approval_tx: tokio::sync::mpsc::UnboundedSender<ApprovalRequest>,
}

#[async_trait::async_trait]
impl imp_core::UserInterface for TuiUserInterface {
    async fn ask(&self, question: &str, options: &[String]) -> Option<usize> {
        let (tx, rx) = oneshot::channel();
        let req = ApprovalRequest {
            question: question.to_string(),
            options: options.to_vec(),
            tx,
        };
        self.approval_tx.send(req).ok()?;
        rx.await.ok().and_then(|r| r.selected)
    }
}
```

```rust
fn handle_approval_key(&mut self, key: KeyEvent) {
    match key.code {
        KeyCode::Up => {
            if let Some(ref mut p) = self.pending_approval {
                p.selected = p.selected.saturating_sub(1);
            }
        }
        KeyCode::Down => {
            if let Some(ref mut p) = self.pending_approval {
                p.selected = (p.selected + 1).min(p.options.len().saturating_sub(1));
            }
        }
        KeyCode::Enter => {
            if let Some(p) = self.pending_approval.take() {
                let _ = p.tx.send(ApprovalResponse { selected: Some(p.selected) });
            }
        }
        KeyCode::Esc => {
            if let Some(p) = self.pending_approval.take() {
                let _ = p.tx.send(ApprovalResponse { selected: None });
            }
        }
        _ => {}
    }
}
```
