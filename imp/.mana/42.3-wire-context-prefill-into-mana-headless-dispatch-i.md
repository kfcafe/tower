## Task

Wire the context prefill assembler into `run_headless_mode()` in `crates/imp-cli/src/main.rs` so that `imp run <unit_id>` automatically preloads referenced files into the agent's cached context.

## Implementation

### 1. Parse `files:` from unit frontmatter

Add optional `files` field to `UnitFrontmatter`:

```rust
struct UnitFrontmatter {
    // ... existing fields ...
    #[serde(default)]
    files: Vec<String>,
}
```

Add a method to `ManaUnit` that produces `Vec<FileSpec>`:

```rust
impl ManaUnit {
    fn file_specs(&self) -> Vec<imp_core::context_prefill::FileSpec> {
        if !self.files.is_empty() {
            // Parse explicit file declarations with optional mode suffixes
            self.files.iter().filter_map(|s| parse_file_spec(s)).collect()
        } else {
            // Auto-detect from description
            imp_core::context_prefill::detect_file_paths(&self.description)
        }
    }
}
```

Parse mode suffixes: `src/foo.rs:tail:50` → `FileSpec { path: "src/foo.rs", mode: Tail(50) }`.

### 2. Assemble and inject in run_headless_mode

After loading the unit, before creating SessionOptions:

```rust
let unit = load_mana_unit(&cwd, unit_id)?;

// Assemble file context from unit references
let file_specs = unit.file_specs();
let assembled = if file_specs.is_empty() {
    imp_core::context_prefill::AssembledContext::empty()
} else {
    let prefill_config = imp_core::context_prefill::PrefillConfig::default();
    let ctx = imp_core::context_prefill::assemble_context(&file_specs, &cwd, &prefill_config);
    for warning in &ctx.warnings {
        eprintln!("[imp] context prefill: {warning}");
    }
    if !ctx.included_files.is_empty() {
        eprintln!("[imp] prefilled {} files ({} est. tokens)", ctx.included_files.len(), ctx.estimated_tokens);
    }
    ctx
};

let options = SessionOptions {
    context_prefill: assembled.messages,
    // ... rest unchanged ...
};
```
