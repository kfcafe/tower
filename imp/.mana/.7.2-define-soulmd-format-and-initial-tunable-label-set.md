  Defined minimal direction for soul.md format and tunables.
  - Relevant OpenClaw lesson: SOUL.md is a real markdown identity document, not disguised config. Its value comes from being readable, editable, and directly embodied by the prompt.
  - Recommended imp shape: keep soul.md freeform overall, but include one builder-managed section with labeled one-line tunables, e.g. a `## Tunables` section containing bullet lines like `- Autonomy: <sentence>`.
  - Each tunable corresponds to exactly one stable line. Parsing stays exact-match only: if the text after the label matches a known generated variant, the builder shows the preset; otherwise it shows edited/mod.
  - The file remains human-first: prose above/below the tunables section is unrestricted and not builder-owned.
  - More than five tunables is good; richness should come from more adjustments, not one control fan-out.

  Initial tunable label set (broad but still simple):
  1. Autonomy — default ask-vs-act threshold.
  2. Brevity — default response length.
  3. Caution — risk posture around changes and uncertainty.
  4. Warmth — interpersonal tone and encouragement level.
  5. Planning — amount of forethought before action.
  6. Candor — how directly uncertainty, disagreement, or limitations are stated.
  7. Thoroughness — how much detail to inspect before concluding.
  8. Initiative — bias toward doing useful work without prompting.
  9. Precision — preference for exact, concrete language.
  10. Playfulness — tolerance for light wit or personality in phrasing.
  11. Deference — how strongly to mirror or yield to user framing versus pushing back.
  12. Skepticism — tendency to challenge assumptions and verify claims.
  13. Teaching — tendency to explain and help the user understand, not just execute.
  14. Decisiveness — willingness to commit to a direction when ambiguity remains.
  15. Formality — casual versus polished/professional tone.

  These labels can power a richer builder without requiring schema complexity because each remains one labeled line in soul.md.
