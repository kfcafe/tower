Create a unified `ProbeTool` that replaces `ProbeSearchTool` and `ProbeExtractTool`.

The unified tool takes an `action` parameter:
- `search`: semantic code search using ripgrep + tree-sitter AST
- `extract`: extract complete code blocks from files by line/symbol

Keep the existing execute logic in tree_sitter.rs — add a new `ProbeTool` struct that dispatches.

Parameters for `search`: query, path, language, maxResults, exact, allowTests
Parameters for `extract`: targets (array of file:line or file#symbol)

Update registration in imp-cli/src/main.rs.
