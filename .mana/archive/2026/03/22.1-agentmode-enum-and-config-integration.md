## What to implement

Add the `AgentMode` enum to `config.rs` with methods that define which tools and mana actions each mode allows. This is the single source of truth for mode permissions.

### AgentMode enum

```rust
#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize, Default)]
#[serde(rename_all = "kebab-case")]
pub enum AgentMode {
    #[default]
    Full,
    Worker,
    Orchestrator,
    Planner,
    Reviewer,
    Auditor,
}
```

### Methods on AgentMode

1. `allowed_tool_names(&self) -> &'static [&'static str]` — Returns the tool names this mode permits.

   | Mode | Tools |
   |------|-------|
   | Full | all (return empty slice to signal "no filtering") |
   | Worker | read, grep, find, ls, scan, web, diff_show, write, edit, multi_edit, diff_apply, bash, mana, ask |
   | Orchestrator | read, grep, find, ls, scan, web, diff_show, mana, ask |
   | Planner | read, grep, find, ls, scan, web, diff_show, mana, ask |
   | Reviewer | read, grep, find, ls, scan, web, diff_show, ask |
   | Auditor | read, grep, find, ls, scan, web, diff_show, mana |

   Note: Worker has everything except it will have mana actions restricted. Full returns an empty slice meaning "allow all" (so shell tools and future tools work without listing them).

2. `allows_tool(&self, name: &str) -> bool` — Convenience check. Full always returns true. Others check the allowed list.

3. `allowed_mana_actions(&self) -> &'static [&'static str]` — Which mana sub-actions this mode permits.

   | Mode | Mana actions |
   |------|-------------|
   | Full | all (empty slice = no filtering) |
   | Worker | show, update, status, list |
   | Orchestrator | all (status, list, show, create, close, update, run) |
   | Planner | status, list, show, create |
   | Reviewer | (no mana access) |
   | Auditor | status, list, show |

4. `allows_mana_action(&self, action: &str) -> bool` — Convenience check.

5. `instructions(&self) -> Option<&'static str>` — Mode-specific behavioral instruction for the system prompt.

### Config integration

Add `pub mode: AgentMode` to `Config` (with `#[serde(default)]`).

Add to `Config::merge()`: override mode if non-default.

Add env override in `Config::resolve()`: `IMP_MODE` env var.
