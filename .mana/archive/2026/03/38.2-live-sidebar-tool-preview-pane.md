Create the right-side sidebar that auto-follows the active tool call.

Implementation:
- Create `imp/crates/imp-tui/src/views/sidebar.rs`
- State: which tool_call_id is being displayed, scroll offset, whether open
- Rendering: title line (tool name + status), content area (scrollable)
- Content source: `ToolOutputDelta` events for streaming, `ToolExecutionEnd` for final
- Auto-follow: when a new tool starts, sidebar switches to it
- When idle: show last tool output, dimmed header

Integration:
- `app.rs`: add `sidebar_open: bool`, `sidebar_tool_id: Option<String>`, `sidebar_scroll: usize`
- Layout: when sidebar open, main area splits into chat(60%) + sidebar(40%, min 30 cols)
- `Ctrl+P` toggles sidebar
- Auto-open: if terminal width ≥ 120 on first `ToolExecutionStart`

Per-tool rendering:
- bash: raw output, streaming
- edit: show as unified diff
- read: file content
- grep: matches with line numbers
- Default: raw tool output text
