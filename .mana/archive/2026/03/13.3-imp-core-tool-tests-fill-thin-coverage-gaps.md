## Modules needing more tests

### tools/edit.rs (currently 3 tests)
Add:
- CRLF line ending preservation (write CRLF file, edit, verify CRLF preserved)
- Multiple occurrences of oldText (should replace first occurrence only, or error — check current behavior and test it)
- Empty oldText parameter → error
- Edit nonexistent file → error with helpful message
- Large file edit (verify truncated diff output)

### tools/write.rs (currently 3 tests)
Add:
- CRLF detection and preservation when overwriting
- Write empty content (should work — creates empty file)
- Write to path with deep nested nonexistent directories
- Write binary-like content (content with null bytes — should still work, write tool writes bytes)

### tools/read.rs (currently 5 tests)
Add:
- Large file truncation (write 3000 lines, read, verify truncated to 2000 with temp file message)
- Read directory (not a file) → error
- Read empty file → returns empty content (not error)

### tools/shell.rs (currently 3 tests)
Add:
- Shell tool timeout behavior
- Missing optional param uses default value from `{param|default}` syntax
- Required param missing → error
- Command with stderr output → included in result
- Truncation mode "tail" vs "head"

### tools/tree_sitter.rs (currently 5 tests)
Add:
- probe_search with language filter parameter
- probe_extract with multiple targets
- scan with directory action parameter
- ast_grep dry run (search without replace)

### tools/bash.rs (currently 5 tests)
Add:
- Command that produces both stdout and stderr (merged output)
- Command with very long output (verify truncation)

### tools/multi_edit.rs (currently 4 tests)

- Empty edits array → error
- Edit with missing path → error
- Edits where later edit depends on earlier edit's change (already tested but add one more complex case)
