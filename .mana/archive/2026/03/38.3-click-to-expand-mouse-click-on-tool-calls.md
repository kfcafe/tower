Add mouse click handling so clicking a tool call line in chat shows its output in the sidebar.

Implementation:
- Track which Y coordinates correspond to which tool calls during render
- Store a `click_map: Vec<(u16, String)>` mapping Y → tool_call_id
- On `MouseEventKind::Down` at a Y in the click_map: set `sidebar_tool_id` and open sidebar
- Need to rebuild click_map each render pass (tool call lines shift as chat scrolls)

Also:
- Click on sidebar area → sidebar gets scroll focus (mouse wheel scrolls sidebar)
- Click on chat area → chat gets scroll focus (mouse wheel scrolls chat)
- Track which pane has focus: `active_pane: Pane { Chat, Sidebar }`

## Requires
- views/sidebar.rs
